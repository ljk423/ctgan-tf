ctgan.data_modules.ConditionalGenerator¶

class ctgan.data_modules.ConditionalGenerator(data=None, output_info=None, log_frequency=None)[source]¶

Conditional Generator.

This generator is used along with the model defined in ctgan.models.Generator, to sample conditional vectors.

Parameters
data: np.ndarray, default=None

Transformed input data by ctgan.data_modules.DataTransformer.

output_info: list[tuple], default=None

List containing metadata about the data columns of the original dataset, namely the number of columns where to apply a given activation function.

log_frequency: bool, default=None

Whether to use log frequency of categorical levels in conditional sampling.

Attributes
n_opt: int

Number of generated features by each conditional vector.

See also

DataTransformer

Transforms the input dataset by applying mode-specific normalization and OneHot encoding.

sample(self, batch_size)[source]¶

Sample a conditional vector for the given batch of data.

For a more detailed implementation of this method, consult section 4.3 of [XSCIV19].

Parameters
batch_size: int

Size of the data batch.

Returns
None, or tuple(np.ndarray, np.ndarray, np.ndarray, np.ndarray)

None, if the training data did not contain any discrete columns. A tuple containing:

  • cond: conditional vector

  • mask: mask vector (the selected column will be 1, the other 0)

  • col_idx: index of the mask vector which is set to 1

  • opt_idx: index of the column vector which is set to 1
