ctgan.data_modules.DataSampler¶

class ctgan.data_modules.DataSampler(data, output_info)[source]¶

Data Sampler.

It samples real data from the original dataset.

Parameters
data: np.ndarray

Training data.

output_info: list[tuple]

List containing metadata about the data columns of the original dataset, namely the number of columns where to apply a given activation function.

See also

DataTransformer

Transforms the input dataset by applying mode-specific normalization and OneHot encoding.

Initialize self. See help(type(self)) for accurate signature.

sample(self, n_samples, col_idx, opt_idx)[source]¶

Sample a batch of training data.

Parameters
n_samples: int

Size of the batch.

col_idx: np.ndarray

If col is None, then there won’t be any restrictions to which data can be sampled. Otherwise, assuming that discrete variables are OneHot encoded, it samples data that has the value of the index opt on column col set to 1.

opt_idx: np.ndarray

If col is None, then there won’t be any restrictions to which data can be sampled. Otherwise, assuming that discrete variables are OneHot encoded, it samples data that has the value of the index opt on column col set to 1.

Returns
Real training data.
