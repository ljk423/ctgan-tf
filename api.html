ctgan-tf API¶

This is the full API documentation of the ctgan-tf toolbox.

ctgan.synthesizer: CTGAN Synthesizer¶

The ctgan.synthesizer module contains the definition of the CTGAN Synthesizer - the “main” class for training a Conditional Tabular GAN, as well as helper methods to control the execution of the training steps.

It contains the Tensorflow 2 implementation of the work published in Modeling Tabular data using Conditional GAN [XSCIV19]. The original PyTorch implementation can be found in the authors’ GitHub repository.

synthesizer.CTGANSynthesizer([file_path, …])

Conditional Table GAN Synthesizer.

ctgan.data_modules: Data modules¶

The ctgan.data_modules module contains the definition of classes used by ctgan.synthesizer.CTGANSynthesizer during training.

These classes mainly provide data transformation and sampling. For further details, please consult sections 4.2-4.3 of [XSCIV19].

Conditional Generator.

data_modules.DataTransformer([n_clusters, …])

Data Transformer.

data_modules.DataSampler(data, output_info)

Data Sampler.

ctgan.models: Neural Network and Transformer Models¶

The ctgan.models module contains the definition of the neural networks used as part of the Generative Adversarial Network built in ctgan.synthesizer.CTGANSynthesizer.

They are built using TensorFlow 2, and make use of the custom layers available in ctgan.layers.

For further details, please consult sections 4.4 of [XSCIV19].

This module also contains the extension of some sklearn models, used in ctgan.data_modules.DataTransformer.

ctgan.layers: Neural Network Layers¶

The ctgan.layers module contains the definition of custom neural networks layers, and helper methods to initialize their weights and bias.

For further details, please consult sections 4.4 of [XSCIV19].

layers.ResidualLayer(input_dim, output_dim)

Residual Layer used on the Generator.

layers.GenActivation(input_dim, output_dim, …)

Conditional Generator activation layer.

Utilities¶

layers.init_bounded(shape, \*\*kwargs)

Initializes the weights or bias of a fully connected layer according to the dim passed in kwargs.

ctgan.losses: Model losses¶

The ctgan.losses module contains the definition of custom loss functions.

For further details, please consult sections 4.3 of [XSCIV19], and WGAN-GP paper [GAA+17].

losses.conditional_loss(cond_info, data, …)

Computes the loss for conditional vectors.

losses.gradient_penalty(func, real, fake[, …])

Calculates the gradient penalty loss for a batch of “averaged” samples.

ctgan.cli: Command line interface¶

The ctgan.cli provides a command-line interface for using the ctgan-tf toolbox.

cli.load_demo()

Loads an external dataset, for testing purposes.

cli.cli()

CTGAN command-line interface.
